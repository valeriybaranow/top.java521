**Задача 1: Расчет суммы и среднего значения**

**Описание задачи:** Напишите программу, которая принимает два числа от пользователя и выводит их сумму и среднее значение.

**Тест ввода:**

Введите первое число: 10

Введите второе число: 20

**Ожидаемый вывод:**

Сумма: 30

Среднее значение: 15.0

**Задача 2: Проверка четности числа**

**Описание задачи:** Напишите программу, которая проверяет, является ли введенное пользователем число четным или нечетным.

**Тест ввода:**

Введите число: 7

**Ожидаемый вывод:**

Число 7 - нечетное.

**Задача 7\*: Проверка на простое число**

**Справка**:

1\. Цикл for перебирает элементы,

Пример записи:

for (int i = 0; i < 10; i++)  {

//your code

}

2\. Цикл while работает пока выполняется условие.

Пример записи:

while (a > b)  {

//your code

}
